De viktigste enhetene i CSS

I denne tutorialen vil du lære om de mest brukte enhetene i CSS, og hvordan du bruker dem for å designe responsive og skalerbare nettsider.

px - Piksler

px står for piksler og brukes til å angi faste størrelser på elementer. En piksel er en enhet basert på skjermoppløsning, og gir presis kontroll over størrelsen på elementer. 

h1 {
    font-size: 24px;
}

rem - Root em

rem refererer til fontstørrelsen på <html>-elementet. Det gjør det enkelt å lage skalerbare design, siden rem er relatert til rotnivå-fontstørrelsen. 

p {
    font-size: 1.5rem; /* 15px hvis html { font-size: 10px; } */
}

% - Prosent

% brukes for å angi størrelser som en prosentandel av det overordnede elementet. Dette er nyttig for responsiv design hvor størrelsen på elementene endres med vinduets størrelse. 

div {
    width: 50%;
}

em - Relative enheter

em refererer til fontstørrelsen på det overordnede elementet. Det er lik rem, men beregnes relativt til det elementet hvor det brukes. 

button {
    padding: 0.5em 1em;
}

vh/vw - Viewport Height/Width

vh og vw er relative til henholdsvis høyden og bredden på visningsområdet. 1vh er 1% av vinduets høyde, mens 1vw er 1% av vinduets bredde. 

.hero {
    height: 100vh;
    width: 100vw;
}
