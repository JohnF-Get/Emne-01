Lage en ramme med CSS

I denne tutorialen vil du lære hvordan du lager en ramme (border) rundt elementer ved hjelp av CSS. Vi vil dekke ulike egenskaper som farge, tykkelse, linjetype og avrundede hjørner.

1. Enkel solid ramme

For å lage en enkel, solid ramme rundt et element, bruker du CSS-egenskapen border: 

div {
    border: 2px solid black;
}

Dette vil gi en solid svart ramme med 2px tykkelse.

Eksempel på en solid ramme

2. Avrundede hjørner

Du kan bruke border-radius for å runde hjørnene på rammen: 

div {
    border: 2px solid black;
    border-radius: 10px;
}

Dette vil gi en ramme med avrundede hjørner.

Eksempel på avrundede hjørner

3. Ulike linjetyper

Det finnes flere typer linjer du kan bruke til rammen, som dashed og dotted: 

div {
    border: 2px dashed red;
}

Dette vil gi en rød, stiplet (dashed) ramme.

Eksempel på stiplet ramme

4. Kombinasjon av farger, tykkelser og linjetyper

Du kan kombinere alle disse egenskapene for å lage unike rammer:

div {
    border: 4px dotted blue;
    border-radius: 15px;
}

Dette vil gi en blå, prikket (dotted) ramme med 4px tykkelse og avrundede hjørner.

Eksempel på kombinert ramme
