Endre forgrunns- og bakgrunnsfarge med CSS

I denne tutorialen vil du lære hvordan du endrer forgrunns- og bakgrunnsfarge ved hjelp av CSS. Farger kan gi nettsiden din liv og gjøre den mer brukervennlig.

1. Endre forgrunnsfarge

Forgrunnsfargen (tekstfargen) endres ved å bruke CSS-egenskapen color. Her er et eksempel: 

p {
    color: rgb(214, 51, 132);
}

Dette setter tekstfargen for alle avsnitt til mørk rosafarge: rgb(214, 51, 132).

2. Endre bakgrunnsfarge

Bakgrunnsfargen endres ved å bruke CSS-egenskapen background-color. Her er et eksempel: 

body {
    background-color: #ffd3d393;
}

Dette setter bakgrunnsfargen til en lys rosafarge (#ffd3d393) for hele nettsiden.

3. Kombinere forgrunns- og bakgrunnsfarge

Du kan kombinere begge egenskapene i samme CSS-regel:

div {
    color: #ffffff;
    background-color: #007bff;
}

Dette setter tekstfargen til hvit og bakgrunnsfargen til blå for alle <div>-elementer.
