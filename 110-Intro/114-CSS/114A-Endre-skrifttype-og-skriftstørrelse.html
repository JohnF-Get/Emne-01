Endre skrifttype og skriftstørrelse med CSS

I denne tutorialen vil du lære hvordan du endrer skrifttype og skriftstørrelse ved hjelp av CSS. Typografi er en viktig del av webdesign og kan enkelt justeres med noen få linjer med kode.

1. Endre skrifttype

Du kan endre skrifttypen for en HTML-side ved å bruke CSS-egenskapen font-family. Her er et eksempel: 

body {
    font-family: 'Montserrat', sans-serif;
}

I dette eksemplet vil hele siden bruke Montserrat som skrifttype. Hvis Montserrat ikke er tilgjengelig, vil den bruke en generisk sans-serif-skrifttype.

2. Endre Skriftstørrelse

For å endre skriftstørrelsen, bruker du CSS-egenskapen font-size. Her er et eksempel: 

h1 {
    font-size: 3rem;
}

Dette eksemplet setter skriftstørrelsen på alle <h1>-elementer til 3rem, som tilsvarer 48px hvis grunnstørrelsen er 16px.

Bruke en relativ enhet for skriftstørrelse

En annen metode jeg ofte foretrekker, er å sette font-size på html-elementet til 62.5%. Dette gjør det enkelt å regne om rem til pikselverdier. 1rem blir 10px, som vist her: 

html {
    font-size: 62.5%; /* 1rem = 10px */
}

body {
    font-size: 1.5rem; /* 15px */
}

Dette gjør det enklere å bruke rem-enheter for å skalere skriftstørrelser, fordi det blir lettere å regne om verdiene direkte til piksler.

3. Kombinere skrifttype og skriftstørrelse

Du kan også kombinere begge egenskapene i samme regel:

p {
    font-family: 'Literata', serif;
    font-size: 1.5rem;
}

Dette setter teksten i avsnittene til Literata med en skriftstørrelse på 1.5rem.
