Endre padding og margin med CSS

I denne tutorialen vil du lære hvordan du endrer padding og margin ved hjelp av CSS. Å forstå forskjellen mellom padding og margin er viktig for å kontrollere avstanden og utseendet på elementene på en nettside.

Forskjellen mellom padding og margin

Padding er avstanden mellom innholdet i et element og elementets kantlinje (border), mens margin er avstanden mellom kanten på et element og nabo-elementene eller ytterkanten av containeren det befinner seg i.

Sette padding og margin med én verdi

Hvis du bruker én verdi, blir denne brukt for alle fire sidene (topp, høyre, bunn, venstre): 

div {
    padding: 20px;
    margin: 10px;
}

I dette eksemplet vil padding være 20px på alle fire sidene av <div>, og margin vil være 10px på alle fire sidene.

Sette padding og margin med to verdier

Hvis du bruker to verdier, setter den første verdien padding/margin for topp og bunn, og den andre verdien for høyre og venstre: 

div {
    padding: 20px 10px;
    margin: 10px 5px;
}

I dette eksemplet vil topp og bunn ha 20px padding, høyre og venstre 10px. Marginen vil være 10px topp og bunn, og 5px høyre og venstre.

Sette padding og margin med fire verdier

Med fire verdier kan du spesifisere padding/margin for hver enkelt side i rekkefølgen: topp, høyre, bunn, venstre: 

div {
    padding: 20px 15px 10px 5px;
    margin: 10px 8px 6px 4px;
}

I dette eksemplet er padding 20px på toppen, 15px på høyre side, 10px på bunnen, og 5px på venstre side. Marginen er 10px topp, 8px høyre, 6px bunn, og 4px venstre.
